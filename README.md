📝 Project: "Wanna Be My Friend?" Waitlist Form
🎯 Overview
This project is a part of The Odin Project's Intermediate HTML and CSS curriculum. The goal was to build a visually appealing and fully functional sign-up form using modern web standards.

✨ Key Features
Custom Validation: Implemented HTML5 validation for names, emails, and phone numbers.

Modern CSS Styling: Utilized CSS Variables (--earthy-pink) for a consistent color theme and text-stroke for a unique header design.

Interactive UI: Added :focus and :invalid pseudo-classes to provide real-time visual feedback to the user.

Mobile Friendly: Used type="tel" to ensure a numeric keypad appears for mobile users.

🛠️ Technologies Used
HTML5 (Forms, Fieldsets, Custom attributes)

CSS3 (Flexbox, Pseudo-elements ::after, Transitions)

Git/GitHub (Version control and deployment)

💡 What I Learned
How to use ::after to automatically add "required" asterisks without cluttering the HTML.

The difference between type="number" and type="tel" for better user experience.

How to create a custom "active" state for input fields using the border-bottom and :focus logic.
